  • Multiplication and division (and)
  • Addition and subtraction (or)
  • Negative signs (not)

  • Multiplication and division (and) happen next
  • Addition and subtraction (or) are the last step

    What are list comprehensions?

    They are a way to transform a list from one format to another. They are a Pythonic alternative to using map or filter

    Transform the following code into a list comprehension: 

        squares = []
    for i in range(10):
        squares.append(i**2)
    print(squares)

    Remember, the format for a list comprehension is: newList = [value loop condition] 

        squares = [i**2 for i in range(10)]   # => Where i**2 is the value, and for i in range(10) is the loop
